## 🚀 Features

* 🎨 **Modern Glowing & Glassmorphic UI/UX**: Implemented interactive pointer-detection spotlight gradients on info-cards, project cards, skill boxes, and certificate cards.
* ⚡ **Ultra-smooth Animations**: Powered by **Framer Motion** for stagger entrance delays and responsive transitions.
* 🌓 **Dynamic Theme Switching**: Seamlessly toggle between dark mode (neon cyan/purple highlights) and light mode (neon blue/pink highlights).
* 🖥️ **Global Cursor Tracking Spotlight**: Custom-designed fixed backdrop glow that tracks mouse coordinates with direct DOM inline styling (60fps+ tracking, zero React re-render lag).
* 💼 **Comprehensive Sections**: Projects, Skills, Certificates, Resume, Blog, and Contact.
* 📱 **Mobile & Desktop Responsive Layouts**: Fully responsive grid systems and flex containers with custom text-wrapping bounds.

## ⚙️ Setup Instructions

To run this project locally:

```bash
# 1️⃣ Clone the repository
git clone https://github.com/nikhileswarb7-dotcom/nikhil-portfolio.git

# 2️⃣ Navigate to project directory
cd nikhil-portfolio

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run development server
npm run dev
```

Now open [http://localhost:3000](http://localhost:3000) or check the output URL in your browser! 🚀

## 🌟 Deployment on Vercel

To deploy this project to Vercel:
1. Make sure to choose the **Vite** framework preset.
2. Under Build Settings, change the **Output Directory** to `build` (since Vite is configured with `outDir: "build"` in `vite.config.ts`).
3. Click **Deploy**.
